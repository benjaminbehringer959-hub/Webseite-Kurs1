# WordPress Setup Checklist (LearnDash + PMPro + Polylang)

## 1) Basis
- [ ] WordPress 6.x installieren (HTTPS aktiv)
- [ ] Theme setzen (leicht, responsive)
- [ ] Grundfarben + Typografie einstellen

## 2) Plugins
- [ ] LearnDash installieren und aktivieren
- [ ] Paid Memberships Pro installieren und aktivieren
- [ ] Stripe + PayPal Gateways fuer PMPro konfigurieren
- [ ] Polylang installieren und aktivieren
- [ ] SEO Plugin (Yoast/RankMath) installieren
- [ ] Cache Plugin installieren

## 3) Mehrsprachigkeit
- [ ] Sprachen: DE (Default), EN hinzufuegen
- [ ] Sprach-Overlay (nur 1. Besuch) definieren
- [ ] Header-Sprachschalter (DE | EN) platzieren

## 4) LearnDash Kurs
- [ ] Kurs 'Mind for Media' anlegen
- [ ] 20 Lektionen Tag 1-20 anlegen
- [ ] Drip-Content: Tag 2 nach 1 Tag, Tag 3 nach 2 Tagen ...
- [ ] Fortschrittsanzeige aktivieren

## 5) Inhalte
- [ ] Tag-Templates in LearnDash einpflegen (Kurztext, Aufgaben, Bonuswissen)
- [ ] Quiz je Tag (3 Fragen) anlegen
- [ ] Reflexionsfragen einfuegen
- [ ] Tag 1 + Tag 20: Konzentrationstest (Quiz-Auswertung)

## 6) Membership / Zugriff
- [ ] PMPro Levels anlegen (Free, Pro)
- [ ] Tag 1-5 frei fuer Free
- [ ] Tag 6-20 nur Pro
- [ ] Upgrade-CTA ab Tag 6 anzeigen

## 7) Seiten
- [ ] Landing Page
- [ ] Onboarding (2-3 Fragen, Schrittanzeige)
- [ ] Vorwort
- [ ] Abschluss
- [ ] Rechtliches (Impressum, Datenschutz, AGB)

## 8) QA
- [ ] Testuser anlegen
- [ ] Onboarding > Vorwort > Tag 1-5
- [ ] Paywall ab Tag 6 pruefen
- [ ] Zahlung testen
- [ ] Tag 20 > Abschluss

## 9) Launch
- [ ] Performance check
- [ ] Mobile check
- [ ] Backup + Updates
